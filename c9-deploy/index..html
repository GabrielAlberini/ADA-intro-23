Deploy

EL deploy es la segunda etapa del desarrollo de un proyecto web. En la etapa de deploy "subimos nuestra carpeta a internet. También es conocido como "subir a producción".

Dominio

El dominio es el nombre que llevará nuestra página web.

La función del dominio es encmascarar el IP de mi sitio web.

Servidor

Un servidor es un servicio que alquilaremos y que va a almacenar nuestra página web.

Servidores pagos: goDaddy, hostinger, donWeb,cloud solutions

Tipo de servidores

  • Gratuito como: Vercel, Github pages, Github pages

Requisitos

  • Una carpeta con archivos web.
  • Repositorio
